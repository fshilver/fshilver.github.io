[명령어] ip

네트워크 인터페이스 목록

~$ ip link

1: lo: <LOOPBACK,UP,LOWER_UP> mtu 65536 qdisc noqueue state UNKNOWN mode DEFAULT group default qlen 1000
link/loopback 00:00:00:00:00:00 brd 00:00:00:00:00:00
2: ens160: <BROADCAST,MULTICAST,UP,LOWER_UP> mtu 1500 qdisc mq state UP mode DEFAULT group default qlen 1000
    link/ether 00:0c:29:d5:de:3f brd ff:ff:ff:ff:ff:ff

네트워크 인터페이스 링크 업/다운

~$ ip link set eth0 up
~$ ip link set eth0 down

IP 할당/삭제

~$ ip addr add 172.16.32.1 dev eth0
~$ ip addr add 172.16.32.1/24 dev eth0
~$ ip addr del 172.16.32.1 dev eth0

가상 IP 할당

  • 실제 ethernet 이름 : eth0
  • 가상 ethernet 이름 : eth0:1
~$ ip addr add 172.16.32.2/24 dev eth0 label eth0:1

라우팅 테이블 추가/삭제

~$ ip route add 172.16.1.1/24 via 192.168.1.55
~$ ip route del 172.16.1.1/24 via 192.168.1.55
