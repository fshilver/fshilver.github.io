Git Submodule

Heesung Yang

November 14, 2020 | 1 minute

howto

Installation

# submodule 확인
~$ git submodule

# submodule 추가
~$ git submodule add <저장소주소> <서브모듈경로>

## submodule download
~$ git submodule init
~$ git submodule update

Update

~$ cd project # git repository
~$ cd sub1    # submodule 경로라고 가정
~$ git pull   # sub1 의 최신 commit 받아오기
~$ cd ..
~$ git add sub1
~$ git commit -m 'update submodule'

# git 1.8 이후
~$ cd project
~$ git submodule update --remote --merge

Remove

~$ SUBMODULE_DIR=themes/ananke/
~$ git submodule deinit -f $SUBMODULE_DIR
~$ git rm -rf $SUBMODULE_DIR
~$ rm -rf .git/modules/$SUBMODULE_DIR
