설치

Official Guide : https://gohugo.io/getting-started/installing/

Mac

# 설치
brew install hugo

기본 사용법

# 사이트 생성
hugo new site quickstart
cd quickstart

# 테마 설치
git init
git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke
echo 'theme = "ananke"' >> config.toml

# 글 생성
hugo new posts/hello-wolrd.md

# 미리보기
# hugo new 명령으로 글 생성 시 draft: true 로 생성되므로 -D 옵션을 붙여서 draft 상태인 글도 보이도록 함
# 웹브라우져를 열어 http://localhost:1313 입력
hugo server -D
